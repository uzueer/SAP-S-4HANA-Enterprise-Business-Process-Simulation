# SAP S/4HANA Enterprise Business Process Simulation

A comprehensive SAP S/4HANA business process simulation demonstrating the implementation of enterprise workflows across Materials Management (MM), Sales & Distribution (SD), and Production Planning (PP) using SAP Fiori Launchpad.

This project was completed as part of the ERP Solutions for Digital Enterprises course, where a real-world enterprise (APGENCO) was analyzed and its business processes were modeled using SAP S/4HANA applications.

---

## Project Overview

The objective of this project was to understand how SAP S/4HANA integrates multiple business functions into a single enterprise platform.

The case study focuses on:

- Procurement
- Inventory Management
- Sales Processing
- Manufacturing
- Production Planning
- Enterprise Master Data
- Organizational Structure

using SAP Fiori applications.

---

## Business Scenario

The case study is based on **Andhra Pradesh Power Generation Corporation Limited (APGENCO)**, which adopted SAP ERP to streamline operations across multiple business locations.

The project analyzes:

- Existing ERP implementation
- Business challenges
- SAP implementation strategy
- SAP Fiori adoption
- End-to-end business workflows
